Merkle Puzzle Challenge:

Flynn and James wanted to communicate securly so 
Flynn generated this list of merkle puzzles with
instructions on how to crack each one (see below).

James cracked a random one and is using that to communicate with Flynn.
You need to find the one they are using so that we can intercept their plans!


Each hash is generated by doing the following: 

a puzzle is made by taking the letters of 'merkle' and making a 10 character string
such as 'lmlrllreke'

a random salt 20 characters long is also made to prevent a birthday attack.

then do the following hash:
sha1(flynn{puzzle:salt})

for example given the entry:
X. salt:ZAkG3Vlno7QOP6y7ZmWd   hash:c99661e710b338439acf327ed3d1b6360433f448  lmlrllreke

you would brute force all possibilities until you find the one that matches the given hash
flynn{mmmmmmmmmm:ZAkG3Vlno7QOP6y7ZmWd}
flynn{mmmmmmmmme:ZAkG3Vlno7QOP6y7ZmWd}
flynn{mmmmmmmmmr:ZAkG3Vlno7QOP6y7ZmWd}
...
...
flynn{lmlrklllll:ZAkG3Vlno7QOP6y7ZmWd}
sha1(flynn{lmlrllreke:ZAkG3Vlno7QOP6y7ZmWd}) = c99661e710b338439acf327ed3d1b6360433f448

Now we have the shared key of 'lmlrllreke' 

They are using one of these keys and it's your job to find which one.

we've discovered that the sha256 hash of the flag they are using is:
01f53bc813eca724ccd0b74e00ed7ea839d7f0cf7f5ada18431f640cd03553f5 

so once you get the flag 'flynn{puzzle:salt}' that hashes to the one in the list (sha1)
and to the above hash (sha256) you'll know you've gotten the right one.


Note:
Given the sha1 hash of something I'm pretty sure it's impossible (without brutefoce) to work out the sha256 hash
without the original thing being hased.
If this is not the case someone pls msg me and I'll update this challenge
